Установка ruby на MacOS:
ruby -v : Версия руби
curl -L https://get.rvm.io | bash -s stable : Установка менеджера версия для ruby
rvm list known: Включение менеджера
rvm install ruby-2.4.4 : Установка ruby
rvm use ruby-2.4.4 --default : Установка данного интерпретатора дефолтным

Установки rails на MacOS:
gem install rails


Краткий экскурс по ruby:

Вывод:
puts "Любая строка. В конце ставит \n"
print "Любая строка. В конце не ставит \n"
p "Любая строка. При выводе обрамляет данную строку кавычками. В конце ставит \n"

Также можно объединять строки:
puts "Любая" + " Строка"

Внутри вывода можно вставлять переменные. Внутри фигурных скобок переменная превращается в строку, поэтому можно использовать методы строки внутри фигурных скобок
print "Hello #{value_name.method_string}"

Объвление функций:

	body_function

end - Окончание функции
