Установка ruby на MacOS:
ruby -v : Версия руби
curl -L https://get.rvm.io | bash -s stable : Установка менеджера версия для ruby
rvm list known: Включение менеджера
rvm install ruby-2.5.2 : Установка ruby
rvm use ruby-2.5.2 --default : Установка данного интерпретатора дефолтным

Установка yarn:
brew install yarn

Установки rails на MacOS:
gem install rails
